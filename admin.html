ℹ️ Як використовувати математичні формули (LaTeX):

Щоб написати формулу в умові задачі, просто візьміть її у знаки долара: $x^2 + y^2 = r^2$. Для великих формул по центру використовуйте подвійні знаки: $$ \frac{a}{b} = c $$.
